Karatsuba算法：大整数乘法的优化算法

引言

Karatsuba算法是一种高效的大数乘法技术，它通过递归地分割输入数并使用较少的基本乘法步骤来计算结果，从而显著减少了计算时间。

算法核心

分割输入数

当计算两个大数A和B的乘积时，首先将每个数分为两部分：

  • A分为A高位和A低位
  • B分为B高位和B低位

假设A和B都是偶数位数，可以表示为：

A = A高位 * 10^(n/2) + A低位
B = B高位 * 10^(n/2) + B低位

其中n是数的总位数。

减少乘法次数

常规乘法需要计算四个乘积：

  1. A高位 * B高位
  2. A高位 * B低位
  3. A低位 * B高位
  4. A低位 * B低位

然而，Karatsuba算法巧妙地利用代数恒等式，只需计算三个乘积：

  • P1 = A高位 * B高位
  • P2 = A低位 * B低位
  • P3 = (A高位 + A低位) * (B高位 + B低位)
